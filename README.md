Instrucciones de Uso: Aplicación de Clasificador de Frutas

Para entrar en la aplicación se entra en el link:

https://classification-fruit.streamlit.app/

Tienes dos formas de uso.

1. Cámara
Con la cámara del dispositivo (computadora, teléfono móvil) apuntar a la fruta que se quiere clasificar y tomar una foto. 

2. Subir archivo
Se puede subir una imagen desde el dispositivo en los formatos .jpg, .png, .jpeg y .webp.

Después de introducir la imagen se mostrará la clasificación a la que pertenece.

En caso de inactividad y al intentar entrar a la aplicación puede que tarde unos momentos en reiniciar la aplicación para que realice su tarea.
